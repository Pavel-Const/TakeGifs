# Сайт по поиску гифок по тегу с api.giphy.com

## Возможности 
 - Ввод одного или нескольких тегов через запятую для поиска
 - Очистка поля ввода и списка изображений
 - Группировать изображения по тегу
 - Валидация поля ввода
 - Вывод hhtp ошибки и ошибки если ничего не найдено
 - При вводе 'delay' отображение 10 рандомных гифок с задержкой 0,5 секунды

### Инструкция по запуску локально
1. Установить npm-зависимости: npm i
2. Запустить проект на локальном сервере для отладки: npm run dev
3. Cобрать проект: npm run build
